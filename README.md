# OWASP-ZAP

## Usage

- In a new machine, run [init.sh](./init.sh) to configure a new environment
- To save a current state before a fresh install, run [savecurrent.sh](./savecurrent.sh)

## Run OWASP-ZAP

``` zsh
docker-compose up
```
